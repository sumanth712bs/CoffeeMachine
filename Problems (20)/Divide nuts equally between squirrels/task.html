Divide nuts equally between squirrels Open on JetBrains Academy

N squirrels found K nuts and decided to divide them equally. Calculate how many nuts each squirrel gets.

Input data format

There are two positive numbers N and K, each of them is not greater than 10000.


Sample Input:
3
14
Sample Output:
4


Memory limit: 256 Mb
Time limit: 1s
