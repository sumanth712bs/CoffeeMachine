Number dividers Open on JetBrains Academy

Write a program that reads an integer number and checks if it is divisible by 2, 3, 5, and 6.

If the number is divided by M, the program should output "Divided by M".

The program should check all the divisors listed above. The order of divisors in the result can be any.

Use the modulo operator % to check whether a number divided by another one.


Sample Input:
6
Sample Output:
Divided by 2
Divided by 3
Divided by 6



Memory limit: 256 Mb
Time limit: 5s
